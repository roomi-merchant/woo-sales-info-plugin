A custom WooCommerce extension that allows you to assign and manage sales labels or text for individual products.

== Description ==

Woo Sales Info is a lightweight plugin that lets you add custom sales text/labels to WooCommerce products.  
It creates a custom table to store product-label mapping and provides an easy interface in the WordPress admin.

**Features:**
* Add and manage sales text per product.
* Assign WooCommerce products directly from a dropdown.
* Store data in a custom database table for performance.
* Simple and clean UI in the admin panel.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/product-label/` directory, or install via the WordPress Plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to **Woo Sales Info** under the WordPress admin menu.
4. Add or update product sales labels.

== Frequently Asked Questions ==

= Will this work with all WooCommerce themes? =
Yes, it should work with any WooCommerce-compatible theme.

= Can I translate the plugin? =
Yes, the plugin is translation-ready. Use a tool like Loco Translate or add your own `.po`/`.mo` files.
